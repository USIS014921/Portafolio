ReservaFacil

ReservaFácil es una aplicación diseñada para la Universidad Gerardo Barrios, esta plataforma simplifica el proceso de reserva de cubículos. Con ReservaFácil, los estudiantes disfrutan de una experiencia sin complicaciones para asegurar espacios de estudio, optimizando así su rendimiento académico.

The Glam App

The Glam App es una innovadora plataforma que conecta a los usuarios con salones de belleza y barberías cercanas a través de un intuitivo mapa interactivo. Además de facilitar la ubicación de estos establecimientos, ofrece una conveniente zona de compras para productos de belleza, brindando una experiencia integral para satisfacer las necesidades de sus usuarios.

SecondLanguage

SecondLanguage es una aplicación diseñada para estudiantes de primer año de inglés. Ofreciendo un enfoque diferente, la app brinda una variedad de ejercicios de escucha, escritura y pronunciación. Con SecondLanguage, aprender inglés se convierte en una experiencia divertida y efectiva, proporcionando a los estudiantes las herramientas necesarias para dominar el idioma de manera entretenida.
